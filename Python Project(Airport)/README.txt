-------------Airport python Project-------------

	-Acesta e un proiect realizat in python in care am gestionat activitatea unui aeroport.
In cadrul lui,am avut disponibilitatea de a creea multiple entitati pentru clasele aeroport si curse.
Cu ajutorul lor am gestionat diferite curse intre aeroporturi intr-un mod similar din viata reala.
In urma creeari entitatilor,care au fost citite dintr-un fisier de tip text,am realizat totodata validari
pentru a ne asigura ca datele sunt introduse corect dupa care am implementat operatii de tip CRUD pentru 
fiecare dintre acestea.La final,am testat de asemenea si aceste operatii intr-un fisier separat de test.
